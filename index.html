확정손익금 · 미실현손익 · 순이익만 n배 적용
로직 집단

배수 및 필터 조건

확정손익금 / 미실현손익 / 순이익에만 적용되고, 익절횟수와 손절횟수는 그대로 유지됩니다.
- 로직 파일은 Total_Result 시트를 우선 사용하고, 없으면 첫 시트를 사용합니다.
- 파일 내부에서 ‘Group Name’ 헤더가 있는 행부터 자동 인식합니다.
- 같은 로직 집단 안에서 그룹명이 같으면 익절횟수 / 손절횟수 / 확정손익금 / 미실현손익 / 순이익을 먼저 합산한 뒤,
  그 다음 확정손익금 / 미실현손익 / 순이익에만 n배를 적용합니다.
- 결과 엑셀은 로직별 2칸 블록 + 로직 사이 1칸 공백 + 부제목 행 + 그룹당 6행 구조로 정리됩니다.
아직 데이터가 없습니다. 왼쪽에서 로직 집단에 엑셀을 업로드하세요.
